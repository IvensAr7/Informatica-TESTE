CSS é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML.

Ele permite que os desenvolvedores controlem a aparência de elementos em uma página da web, como cores, fontes e layout.

Com o CSS, é possível aplicar estilos a elementos individuais ou a grupos de elementos, tornando a criação de layouts complexos mais fácil e eficiente.
